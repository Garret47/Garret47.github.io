CSS(Cascading Style Sheets) - markup language used to visually design websites.
Objects located on the page are placed using HTML. But CSS is responsible for how these objects look. Their size, color, background image, degree of transparency, position relative to other elements, hover behavior, visual change of buttons when pressed.

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.
"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat.
JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles.
